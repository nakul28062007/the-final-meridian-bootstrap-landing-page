# The Final Meridian 🏔️

A fictional luxury resort website set in Antarctica, built as a self-practice 
project while learning full-stack web development.

## 🌐 Live Demo
[View on GitHub Pages](https://nakul28062007.github.io/the-final-meridian-bootstrap-landing-page/)

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- Bootstrap 5.3
- DaVinci Resolve (hero video editing)

## 📄 Sections
- Hero — fullscreen background video with overlay text and booking bar
- Hotel Features — image cards with CSS overlay gradient
- Surrounding Attractions — Bootstrap carousel with 9 Antarctica locations
- Events — card layout for mindful explorer experiences
- Pricing — three-tier pricing plans
- Contact — address, phone, email with SVG icons
- Footer — navigation links
